### Project Overview:

This project uses the ESP32 in SoftAP / Access Point mode to create a Wi-Fi captive portal where:

1. Users connecting to the ESP32 Wi-Fi automatically see a browser popup captive portal.


2. The captive portal displays an image slider / billboard-style slideshow, fetching images from SD card or SPIFFS storage.


3. The project is adaptable for multiple use cases:

Shop / Event / Marketing / Emergency / Prank

Optional BLE / QR auto-redirect features

### Project Workflow:

1. ESP32 starts in SoftAP mode


2. Captive portal redirects users using DNS hijacking


3. Web page served with HTML + CSS + JS slider


4. Image slider fetches images from SD card / SPIFFS


5. Users interact, images and messages display in slideshow format


6. Optional BLE or QR triggers enhance automation
